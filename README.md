# FF14BookCoverGenerator

ファイナルファンタジーXIV（FF14）のキャラクター紹介カードを、「本の表紙＋帯」風のおしゃれなデザインで手軽に作成・保存できるWebジェネレーターです。

---

## ✨ 主な機能

- **本の表紙をイメージした縦長デザイン**:
  - 600px × 900px（アスペクト比 2:3）の本格的な書籍カバー風レイアウト。
- **文字配置のカスタマイズ**:
  - タイトル（キャラクター名）の「縦書き」と「横書き」を瞬時に切り替え可能。
  - タイトルとデータセンター名の表示位置を、3×3（9分割）のグリッドから選択して自在に配置できます。
  - タイトルやキャッチコピーのフォントサイズもスライダーで微調整可能。
- **帯（オビ）によるキャラクタープロフィール記述**:
  - **左カラム**: プレイスタイル（複数選択可、縦並び）と活動時間。
  - **中央カラム**: 大キャッチコピー、サブキャッチコピー、あらすじ（改行対応）。
  - **右カラム**: ジョブバッジ（複数選択可、メインジョブには「★」を表示）。
- **画像のアップロードとズーム調整**:
  - カバー画像にお好みのキャラクター画像を当てはめ、ズームスライダーで最適な範囲に拡大・縮小が可能です。
- **デザインプリセットとカラーカスタマイズ**:
  - 「文芸」「ダーク」「エレガント」「ファンタジー」「ミニマル」の5つのプリセットを用意。
  - 表紙と帯の背景色、文字色、アクセントカラー、キャッチコピー色を個別設定して、お好みのテーマを作成できます。
  - ドロップシャドウ、テキストシャドウ、境界ボーダーのオン/オフも切り替え可能。
- **画像保存機能**:
  - 「カードを保存」ボタンを押すだけで、デザインされた本風の画像が即座にPNGフォーマットでダウンロードできます（GitHub Pagesなどのホスト環境でも動作します）。

## 🚀 ローカルでの実行方法

本プロジェクトは静的ファイルのみで構成されているため、特別なビルド環境は不要です。
ファイルをダウンロードし、簡易サーバーを起動するだけで動作します。

### Pythonを使用する場合
```bash
# プロジェクトディレクトリに移動
cd FF14BookCoverGenerator

# HTTPサーバーの起動（Python 3）
python3 -m http.server 8080
```
ブラウザで `http://localhost:8080/` にアクセスして動作を確認してください。

### VS Code (Live Server) を使用する場合
VS Codeの拡張機能「Live Server」をインストールし、`index.html` を右クリックして「Open with Live Server」を選択するだけで即座に起動できます。

## 🌐 GitHub Pages での公開手順

1. このリポジトリを自身の GitHub アカウントにフォークまたは作成します。
2. リポジトリの **Settings** (設定) タブを開きます。
3. 左メニューの **Pages** を選択します。
4. **Build and deployment** の Source にて `Deploy from a branch` を選択します。
5. Branch に `main` (または `master`) の `/ (root)` を指定して **Save** します。
6. 数分後、`https://<ユーザー名>.github.io/FF14BookCoverGenerator/` にて公開されます。
